# Eurostat project 
### EU penetentiary statistics top 4 countries
This project uses data provided by Eurostat, to provide information on Penetentiary Occupation percentages across top 4 European countries by GDP.
Namely the countries considered are:
1. Netherlands
2. France
3. Italy
4. Germany

The dashboard provides Occupancy % ages by country, accessible by hovering over the country location
Additionally by clicking any country, additional information is provided:
- Prison employee to inmate ratio
- Bribery convictions

Check out the basic functionality of the dashboard here -> https://019c86dd-7f13-d986-0e29-ea9c1e5a2b9f.share.connect.posit.cloud/

This project is an exercise on deploying interactive web based graph over GeoJSON map, through Shiny.
The dashboard was coded on python. 

### Recreating the dashboard

To recreate the dashboard pull the repository then install the available files:
pip isntall requirements.txt

Post that you can run the code through a bash terminal using:
python app_test.py
OR
If you use VS Code then by installing the Shiny extension run the code within the VS environment
